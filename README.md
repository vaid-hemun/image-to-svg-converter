# Image to SVG Converter

A full-stack web tool to convert PNG/JPG images into clean SVG using Node.js, Express, Potrace and Tailwind CSS.

Inspired by tools like Vectorizer.ai.

---

## Features

- Upload image and convert to SVG
- Live preview (Original vs SVG)
- Adjustable threshold
- Loading animation
- File type validation
- File size limit (5MB)
- Auto cleanup of uploads
- Timeout protection

---

## Tech Stack

- Node.js
- Express.js
- Potrace
- Tailwind CSS
- HTML / JavaScript

---
